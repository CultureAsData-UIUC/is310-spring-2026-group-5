Hi Tara,
This is an interesting topic. I think it is a good choice to study how AI is perceived by reddit users rather than trying to measure direct correlations.

The data you have collected so far is a good start. I appreciate that you are including information about which LLM or AI tool is being discussed as well as the links for every page you are collecting.

I have a few suggestions for moving forward into your final project:

1. There are a few issues with your dataset that need to be corrected. You did not include any headers or titles for your columns so it is not clear what many of your columns mean. There are also some data entry issues. For example, in the column collecting data about which tools were mentioned you have both 'not specified' and 'n/a'. Do these represent two different things or do they need to be standardized?
2. There is not a lot of information about the actual posts in your dataset at the moment. It would be possible to do sentiment analysis on reddit posts at scale, but you are not currently collecting any information to do sentiment analysis on. If you want to know about what students on reddit are saying about AI, it would be helpful to collect some text from the posts as well. There are also other metrics on reddit that may be useful for you to collect to provide context such as the number of upvotes and comments or the date the post was made. (You don't have to get exactly what I am listing here, just some suggestions.) Your observations about the reddit posts seem interesting, but the patterns you are outlining are not clear from the data you have collected.
   - On a similar note, are you determining sentiment of the posts based on title only, or are you taking the entire text into account? You should provide more detail on how you made these determinations. Were there any cases where the sentiment was more difficult to determine? How did you handle these situations?
3. For next steps, you can use the reddit API but have you went through the application process? I am not sure how long that process takes so if you want to scale your project using the API you need to be sure you can access it in time.

I think you are on the right track and if you fill out your dataset you will have some really interesting cultural data to learn from here!
